# 🔐 Caesar Cipher CLI Tool

A Python-based command-line tool that implements the Caesar Cipher algorithm for encrypting and decrypting text. Built as part of my cybersecurity learning journey to understand foundational cryptographic techniques.

---

## 🚀 Features

- Encrypt and decrypt messages using a shift-based algorithm
- Supports uppercase and lowercase characters
- Preserves numbers and special characters
- Input validation for robust execution
- Interactive CLI interface

---

## 🧠 Concepts Demonstrated

- Classical Cryptography (Caesar Cipher)
- Symmetric Encryption Basics
- ASCII Character Manipulation
- Modular Arithmetic in Programming

---

## 🛠️ Tech Stack

- Python 3
- CLI (Command Line Interface)
